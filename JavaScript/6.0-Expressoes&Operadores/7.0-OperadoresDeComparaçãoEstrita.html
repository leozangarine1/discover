Operadores de comparação estritamente igual e estritamente diferente

Os Operadores de comparação estritos comparam o valor e o tipo do objeto 


            let one = 1
            let two = 2

            // === estritamente igual a
            
            console.log( one === "1") 
            false, pois 1 é um number e "1" é uma string

            console.log( one === 1)
            true, pois ambos são tipo number

            // !== estritamente diferente de

            console.log( two !== "2")
            true, pois 2 é number e "2" é uma string, logo, são diferentes.

            console.log( two !== 2) false, pois ambos são tipo number, logo, são iguais.
