Operadores lógicos

- 2 valores booleanos, quando verificados resultará em verdadeiro ou falso 


              let pao = true
              let queijo = false
        
              // AND &&
              console.log(pao && queijo) false
              possui pão E queijo? não, apenas pão é verdadeiro  

              // OR || 
              console.log( pao || queijo) true
              possui pão ou queijo? sim, possui pão
        

              // NOT !
              console.log(!pao) false
              o operador NOT! alterá o valor de true para false e vice-versa
