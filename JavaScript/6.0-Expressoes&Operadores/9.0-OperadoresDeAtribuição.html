Operadores de atribuição

. Quando falamos em atribuição, queremos na verdade dizer assignment, que já vimos bastante, mas podemos também realizar diferentes tipos de atribuição, usando operadores aritméticos. Exemplos: 

            
                // Operadores de atribuição (Assignment)
                let x
            
                // assignment normal:
                x = 1
            
                // addition assignment (adição):
                 x += 2
                (x = x + 2)
    
            
                // subtraction assignment (subtração):
                 x -= 1
                (x = x - 1)
            
                // multiplication assignment (multiplacação):
                 x *= 2
                (x = x * 2)
            
                // division assignment (divisão):
                 x /= 2
                (x = x / 2)
            
                // exponetiation assignment (exponenciação):
                 x **= 2
                (x = x ** 2)
            
                // remainder assignment (resto de divisão):
                 x %= 2
                (x = x % 2)
