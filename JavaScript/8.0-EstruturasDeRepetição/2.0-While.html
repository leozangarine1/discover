While

O loop while mantém o loop através de um bloco de código enquanto a condição for verdadeira

Sintaxe:


              while(condição) {
                  // bloco de código a ser executado
              }

No exemplo a seguir, o código no loop rodará, denovo e denovo, enquanto a variável (i) for menor que 10:


            let text = "";
            let i = 0;
            while(i < 10) {
                text +=console.log(`The number is ${i}`);
                i++;
            }
