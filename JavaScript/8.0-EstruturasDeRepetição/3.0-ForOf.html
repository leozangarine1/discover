For of

O loop for...of percorre objetos iterativos (incluindo Array, Map, Set, o objeto arguments e assim por diante), chamando uma função personalizada com instruções a serem executadas para o valor de cada objeto distinto.

Sintaxe:


            for (variavel of iteravel) {
                declaração
            }

variável: A cada iteração, um valor de uma propriedade diferente é atribuído à variável.

iterável: Objeto cujos atributos serão iterados

Exemplos:


Iterando um Array


            let iterable = [10, 20, 30];

            for (let value of iterable) {
                console.log(value);
            }
            // 10
            // 20
            // 30

Iterando sobre uma String


            let iterable2 = "boo";

            for (let value2 of iterable2) {
                console.log(value2)
            }
            // "b"
            // "o"
            // "o"
