# Scope

* Escopo determina a visibilidade de alguma variável no JS



# Block Statement

- Cada bloco é formado pelas chaves de início { e fim }, o bloco, também criará um novo escopo. Chamamos de block-scoped

            
                // var é global e, também local
            
                console.log(' > existe x antes do bloco? ', x)
            
                {
                    var x = 0
                }
            
                console.log(' > existe x depois do bloco? ', x)
            
                //hoisting

Resultado do console:
> existe x antes do bloco? undefined

> existe x depois do bloco? 0


           //const e let são locais e só funcionam no escopo onde foi criada

           let y = 2 //escopo global
            
           {
               let y = 0 // escopo de bloco
               console.log (y)
           }
        
           console.log (y)
        
 
Resultado do console: 
           0 // console foi printado no bloco
           2 // console pegou o valor de y na escala global
