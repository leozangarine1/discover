## Para criar variáveis

* JS é case-sensitive (sensível à letras maiúsculas e minísculas

* JS aceita a cadeia de caracteres Unicode (caracteres especiais e acentuações)


- Posso:

* Iniciar com esses caracteres especiais: S_

* Iniciar com letras

* Colocar acentos

* Letras maiúsculas e minúsculas fazem diferença


- Não posso:

* Iniciar com números

* Colocar espaços vazios no nome


- Ideal:

* Criar nomes que fazem sentido

* Que explique o que a variável é ou faz

* camelCase ( UsarLetrasMaiúsculasNoLugarDoEspaço)

* snake_case (Colocar_um_undeline_ao_invés_do_espaço)

* Escrever em inglês
