<input type="color" />

- Interface para selecionar cor

- Color picker

# Atributos

- value: RGB

Se inválido, o preto será exibido

- list

* o id de uma tag <datalist> que está no mesmo documento

* <datalist> irá conter uma lista de valores pré definidos a fim de que o usuário escolha uma cor

* Os valores do <datalist> que não forem compatíveis mostrará a cor preta
